yuzUI created by yuzu health

Retro style components built purely with css. Github repo here. Powers yuzu.health and yuzu.party
Inspired by blocks.css

Just add this stylesheet to your html.
<link rel="stylesheet" href="https://unpkg.com/@yuzu-health/yuzui@1/dist/all.min.css" />
Since it's pure CSS, it works with Svelte, React, Vue, regular HTML sites etc...
IN DEVELOPMENT! Proceed at your own caution. Very finicky on mobile (works on yuzu.party but not here for unclear reasons).

Buttons

Works with anything
<button class="yuzui">Click me!</button> <div class="yuzui button">Works with anything</div> <button class="yuzui pressed">Pressed</button> <button class="yuzui loading">When Loading</button>
